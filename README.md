# Vue Free Website

Starter site built with Vue CLI, Vue Router, and GitHub Pages deployment.

## Project setup

```bash
npm install
```

### Run locally

```bash
npm run serve
```

### Build for production

```bash
npm run build
```

## Custom domain

Update `public/CNAME` with your real domain before deploying.

## GitHub Pages

Push to the `main` branch to trigger the workflow in `.github/workflows/deploy.yml`.
